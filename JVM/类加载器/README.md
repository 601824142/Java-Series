# 类加载器

类的加载是需要类加载器完成的，但是类加载器在 JVM 中的作用可不止这些。在 JVM 中，一个类的唯一性是需要这个类本身和类加载一起才能确定的，每个类加载器都有一个独立的命名空间。

- 将类的字节码文件从 JVM 外部加载到内存中
- 确定一个类的唯一性
- 提供隔离特性，为中间件开发者提供便利，例如 Tomcat
